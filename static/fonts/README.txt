Place KrutiDev010.ttf in this folder.

You can download it from:
  https://www.indiatyping.com/index.php/download/krutidev-font

Or extract it from any Windows system that has Kruti Dev installed:
  C:\Windows\Fonts\KrutiDev010.ttf

The font file is required for the spreadsheet editor to display
Krutidev text correctly in the browser.

If the font file is missing, the browser will fall back to a
system serif font and Krutidev text will appear as Latin characters.
